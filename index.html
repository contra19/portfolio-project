About Me

My name is Rob Wisniewski and I am an aspiring web designer. I came to enjoy web design after using templates to create my own websites on wix.com. After making a few sites there with templates, I thought I would take my learning to the next level and learn to build websites from scratch. This lead me to seek out a bootcamp to help build my skills. I came across the UNC Coding Bootcamp, in which I am currently enrolled. This is the doorway I am using to be able to get myself in to web design.

This page is an ode to my projects. Currently there are only 2, but as I grow my portfolio this is where things will live that I have created. I hope you are able to find something you enjoy on this page. My contact details are at the bottom if you would like to know more or approach me about any projects.
